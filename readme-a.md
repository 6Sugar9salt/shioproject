# Character Prompt Generator (A-chan)

#### 📌 About This Tool

「Aちゃん (Gem A)」は、キャラクターの画像や設定画から、動画生成AI「Google Veo 3.1」に最適化された**「外見定義ファイル（Visual Core JSON）」**を生成するアーキテクトツールです。 クリエイティブな「雰囲気」を、AIが解釈可能な「構造化データ」へと変換し、動画生成時のキャラクターの一貫性を担保します。

#### 🚀 How to Use

**STEP 1: API Keyのセットアップ** Google AI Studioで取得したGemini APIキーを「**1. Google Gemini API Key**」に入力し、「読み込み」ボタンを押してください。 _(※セキュリティ仕様: APIキーはブラウザに保存されず、ページを閉じるとリセットされます)_

**STEP 2: モデルの選択** 「**2. Select Model**」で使用するAIモデルを選択します。

- **Gemini 3.0 Pro:** 2025年11月にリリースされた最新フラッグシップモデル。最高峰の推論能力と視覚理解力を持ち、キャラクターの微細なニュアンスまで正確に言語化します。（推奨）
    
- **Gemini 2.5 Flash:** 処理速度と精度のバランスに優れたモデル。素早く結果を確認したい場合や、大量のパターンを試行する場合に適しています。
    

**STEP 3: 画像のインポート** 「**3. Character Image**」のエリアに、キャラクターの立ち絵や三面図をドラッグ＆ドロップします。 _(Point: 全身が描かれている画像を使用すると、より正確な定義が生成されます)_

**STEP 4: コンテキストの補完 (任意)** 画像だけでは伝わらない情報（正確な性格、隠れているパーツの設定、名前など）があれば、「**4. Additional Context**」に入力してください。AIがこれを加味して解析します。

**STEP 5: 生成と保存** 「**解析してJSONを生成**」ボタンをクリックします。 生成されたコードは「**Output JSON**」に表示されます。「Copy JSON」でクリップボードにコピーするか、「Download .json」でファイルとして保存し、制作フローの次工程で使用してください。

#### 💡 System Logic

このツールは、入力された画像がラフスケッチや2Dイラストであっても、出力時には**「High-fidelity 3D rendered style（高精細な3Dレンダリングスタイル）」**として再定義するように設計されています。また、アニメーションの妨げにならないよう、元画像の「ポーズ」情報は意図的に除外し、ニュートラルな立ち姿として記述を作成します。
